# StrideX AI: Persona-Driven Mockup Ecosystem

This directory contains a suite of high-fidelity, PWA-ready HTML mockups for the StrideX AI Cricket Coaching Platform.

## 🚀 How to Launch
1. Open [mockups/index.html](./index.html) in any modern web browser.
2. Select a persona from the landing page to enter their specific portal.

## 👥 Personas
- **Coach Portal**: Manage technical analysis, session scheduling, and student progress.
- **Student App**: Upload videos for AI biomechanical feedback, book sessions, and track skills.
- **Parent Portal**: Manage family accounts, allocate credits, and view child reports.
- **Admin Panel**: Platform governance, club verification, and financial monitoring.

## 🛠️ Technical Stack
- **Structure**: Semantic HTML5.
- **Style**: Minimalist Vanilla CSS with soft shadows and Apple/Google aesthetics.
- **PWA**: Manifest and Service Worker included for standalone installation.
- **Logic**: Vanilla JS for UI transitions and mock interactions.

## 📐 Design Tokens
- **Primary Accent**: #007AFF (Apple Blue).
- **Background**: #f5f5f7 (Clean Light Slate).
- **Radius**: 12px.
- **Typography**: Inter (Google Fonts).
